## Local Setup & Installation
* Clone the repo
```
$ git clone https://github.com/aviralj02/Connectify-MERN.git
```
* Install NPM Packages (use both in /client and /server)
```
$ npm install
```
* Run Project in localhost
```
$ cd client
$ npm run start
```
